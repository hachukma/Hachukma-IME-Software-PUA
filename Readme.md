Hachukma Script IME

Hachukma Script IME is an input method and font package for the experimental Hachukma writing system.

Important Notice

* This project uses Unicode Private Use Area (PUA) codepoints.
* The encoding is experimental and is not part of the Unicode Standard.
* Documents created with this IME require the accompanying font to display correctly.
* Future versions may use different codepoints if the script is later proposed for Unicode encoding.

Script Status

The Hachukma script is currently in the development and presentation stage.

* The script has not been submitted to or approved by the Government of Tripura, the Government of India, or the Unicode Consortium.
* The project is intended for research, testing, cultural presentation, and script development purposes.
* Future development may include refinement of character assignments, encoding methods, and script standards.

Installation

1. Download the latest release.
2. Install the included font files.
3. Install the Hachukma Script IME.
4. Restart applications if necessary.
